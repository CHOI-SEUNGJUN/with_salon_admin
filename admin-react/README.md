# WITH-SALON ADMIN FRONTEND

## Requirements

Node.js  
NPM or YARN  

\+ Docker _(if you want)_

## Installation

```bash
git clone # or git pull
```

in the directory

```bash
npm install
```

## RUN

```bash
npm start
```

open your browser and go to `http://{YOUR_URL}:8080/` to check the dashboard statement.

admin id : with
admin pw : lining

### Using DOCKER

It uses `docker-compose.yml` to create a container.  

It must need `Docker` on your machine first.

```bash
docker-compose up -d --build
docker ps # to check the state
```
